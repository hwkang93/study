# Adapter

**호환되지 않는 두 개의 인터페이스를 연결하는 역할**을 하는 클래스를 Adapter 라고 한다.
그리고 그 Adapter 클래스를 이용해 어떤 문제를 해결하는 방법을 Adapter Pattern 이라고 한다.

- 외부 라이브러리 등의 구성 요소를 재사용하고 싶지만 현재 구현된 소스코드로는 호환이 불가능한 경우
-
