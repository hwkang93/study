# Facade

서브 모듈들에 대한 간략한 인터페이스를 제공하고자 할 때 사용하는 디자인 패턴

Wrapper 가 특정 인터페이스를 준수해야 하며, 폴리포픽 기능을 지원해야 할 경우에는 Facade 패턴보다는 Adapter 패턴을 사용해야 한다.
쉽고 단순한 인터페이스를 제공하고 싶을 경우에 Facade 패턴을 사용하는 것이 좋다.
