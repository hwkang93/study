#IntelliJ 윈도우 단축키

| 단축키              | 설명                                            |
|------------------|-----------------------------------------------|
| ALT + INSERT     | 코드 삽입(Constructor, getter/setter, toString 등) |
| CTRL + ALT + O   | Import 정리(사용하지 않는 Import 삭제)                  |
| CTRL + ALT + V   | 변수 자동 생성                                      |
| CTRL + ALT + L   | 자동 정렬                                         |
| CTRL + SHIFT + R | 모든 파일에서 문자열 변경                                |
| CTRL + SHIFT + F | Find in Files                                 |
| CTRL + B         | 구현체 찾기                                        |
| CTRL + D         | 줄 복사                                          |
| CTRL + R         | 현재 파일에서 문자열 변경                                |
| CTRL + U         | 변수 대문자 or 소문자로 변경                             |
| CTRL + Y         | 커서가 올라가있는 라인 삭제                               |
| SHIFT + SHIFT    | 파일 찾기                                         |
| sout             | System.out.println                            |
| get or set       | getter or setter                              |
| psvm             | public static void main(String[] args)        |
| F7               | (Debugging) 현재 브레이크된 라인에서 실행하고 있는 메소드로 이동     |
| F8               | (Debugging) 현재 브레이크된 라인에서 다음 라인으로 이동          |
| SHIFT + F8       | (Debugging) 브레이크된 라인에서 호출한 곳으로 이동             |
| ALT + F8         | (Debugging) 브레이크된 라인에서 사용 가능한 모든 코드를 실행       |
| ALT + F9         | (Debugging) 포커스 되어있는 라인으로 이동                  |
