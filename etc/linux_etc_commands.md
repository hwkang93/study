# 리눅스 기타 명령어 정리

리눅스 간단한 명령어 정리

| 명령어                | 설명                      | 비고                  |
|--------------------|-------------------------|---------------------|
| yy                 | 현재 줄 복사하기               | yank                |
| y10                | 현재 줄부터 10줄 아래까지 복사      |                     |
| p                  | 마우스 커서 아랫줄에 복사한 내용 붙여넣기 | paste               |
| /[단어]              | [단어] 검색                 |                     |
| cw                 | 마우스 커서가 있는 단어 수정하기      | change word         |
| dd                 | 현재 줄 삭제                 | delete              |
| d10                | 현재 줄부터 10줄 아래까지 삭제      |                     |
| u                  | 실행 취소                   | undo                |
| alias -p           | 별칭 목록 조회                |                     |
| grep -r '문자열' 디렉토리 | 문자열이 포함된 파일 조회          | ex) grep -r 'dev' . |
