# 마크다운 문법 정리

자주 사용하는 마크다운 문법을 정리한 것이다.

## 제목

### 사용 예시

```
# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5
```

### 결과

# 제목 1
## 제목 2
### 제목 3
#### 제목 4
##### 제목 5

## 목록

```*```, ```+```, ```-``` 문자를 지원하며 어떤 문자열을 사용하던 동일한 결과를 나타낸다.
뎁스(depth)는 Tab 으로 구분한다.

### 사용 예시

```
- 과일
    + 사과
    + 바나나
    + 복숭아
        * 물복
        * 딱복

- 소프트웨어 엔지니어
    - 백엔드
    - 프론트엔드
    - DBA
```

### 결과

- 과일
    + 사과
    + 바나나
    + 복숭아
        * 물복
        * 딱복

- 소프트웨어 엔지니어
    - 백엔드
    - 프론트엔드
    - DBA

## 문장

### 사용 예시

```
**강조**

~~취소선~~

_기울게_
```

### 결과

**강조**

~~최소선~~

_기울게_

## 인용구

### 사용 예시

```
> 인용구입니다.
> 
> 내부에 다른 마크다운 문법 사용이 가능해요!
> 
> ```java
> public class Test {
>   public static void main(String... args) {
>       System.out.println("Hello World!");
>   } 
> } 
> ```

### 결과

```
> 인용구입니다.
> 
> 내부에 다른 마크다운 문법 사용이 가능해요!
> 
> ```java
> public class Test {
>   public static void main(String... args) {
>       System.out.println("Hello World!");
>   } 
> } 
> ```
